情境演练培训小结

2021-03-22

上周末两天参加了公司组织的情境演练培训，主要内容是解决管理困境，进行干部管理的实操和情境辅导。结束之后，仍有意犹未尽之意，整理相关笔记如下。

1. 课程纲要

  1. 开场与规则
  2. 演练基础
  3. 情境演练
  4. 总结与行动

2. 学习的三个区域

  • 舒适区
  • 拉伸区
  • 恐慌区

不能在舒适区太久，要适当拉伸，但要避免恐慌，不断学习成长

3. DIE认知模型

  1. Description 描述

    我看、听、摸、闻、尝到…

    选择性观察

    所见未必全信

    所信必然可见

  2. Interpretation 解读

    我猜、认为…

    讲故事

    与经历相关

    有色眼镜

  3. Evaluation 评判

    我判断…

    引发情绪

    与价值观相连

    关注自我批判

4. 搭桥

  1. 重复对方的话或关键词
  2. 讲出对方情绪
  3. 问个问题
  4. 认可、同理
  5. 我也有过类似经历…

5. 事实 vs 观点

事实：

  • 老
  • 每次
  • 总

观点：

  • 经常
  • 从来

6. 你可以这样说

  • 我听到你说
  • 我同意你说的..
  • 你说的…我是能理解的…
  • 我们有共识的是…分歧是…
  • 我理解-下,你的观点点是…
  • 截止目前，事实的部分是…
  • 我建议我们先停下，我自己有点情绪…
  • 我们坐下说…

7. R3低绩效员工面谈

  1. 开场 - 搭桥
  2. 中间 - 沟通细节、倾听
  3. 结尾 - 回顾确认，达成共识

8. 鸡尾酒会

  1. 昨天我学了/做了…
  2. 昨天我喜欢/不喜欢/印象深…
  3. 昨天我的一个启发/思考/发现…
  4. 课后我可能会…

9. 内向 vs 外向

  • 尊重理解差异：井井有条、随遇而安
  • 内向、外向
  • 加大自己的宽度、弹性空间
  • 先认可…但是…

10. F4模型

  1. Facts 事实
  2. Feeling 感受
  3. Finding 启发
  4. Future 行动

输入 -> 感性 -> 理性 -> 输出

11. IIY模型

  1. I 我

    看到、听到(观察的行为)…

  2. I 我

    觉得…

    我的解读

    我自己的感受

    该行为对我产生的影响

  3. You 你

    你怎么想/能理解吗？

    觉得呢？

12. 会议

会议的目的

  • 沟通共识
  • 解决问题
  • 制定政策
  • 集思广益
  • 安排工作

针对不同目的，分别制定策略

13. 向上管理

  • 你与上级的关系，你承担>50%的责任
  • 上级发展好，你受益的可能更高
  • 如果你与上级发生矛盾，组织更大可能向着你的上级
  • 你无法改受上级，但可以影响
  • 了解上级
  • 用支持的方式影响上级
  • 不要看不起上级
  • 不要让上级太意外
  • 让自己育原则、有底线、有价值

14. 其他

  • 谈话陷阱 & 倾听的坑：晚点给建议
  • 目标：事与人
  • 先处理情绪，后处理事情
  • 你有什么建议给到我？
  • R4员工发展：寻找下一个R1/R2方向
  • 多鼓励下属：无成本激励，多找出闪光点

15. 个人评价

  • 温柔
  • 柔和
  • 积极
  • 正直
  • 成熟
  • 稳重
  • 专业
  • 态度好
  • 同理心
  • 冷幽默
  • 工作负责
  • 有一说一
  • 思考建议
  • 乐观严谨
  • 逻辑思维
  • 谦谦君子

16. 个人感悟

  • 团队成员间彼此拉近距离，加深了解
  • 进一步认识自己，了解自己的优点不足
  • 工作向下及向上管理的一些理论及实践基础
